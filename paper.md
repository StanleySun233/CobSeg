# 边界感知双流融合的有监督对话话题分割

---

## 摘要

对话话题分割（Dialogue Topic Segmentation, DTS）是对话理解的基础任务，旨在将多轮对话切分为语义连贯的话题片段。现有有监督方法普遍依赖句级嵌入序列，忽略了话语内部词级边界线索，且对序列中所有话语一视同仁，无法自适应聚焦于边界信息量更高的关键位置。本文提出 **DUD**，其中第一个 **D** 表示 **Dual**，**U** 表示 **UBIW**，第二个 **D** 表示 **AE/BD dual-channel**，包含三项相互协同的核心设计：（1）边界感知双流融合架构，并行建立句级语义流与词级边界流，以边缘门控池化放大话语头尾 token 贡献，以起始/终止双头联合建模边界前后依赖；（2）话语边界信息量自适应加权机制（UBIW），以中心化残差放缩在无额外监督下自适应增强高边界信息量话语；（3）AE/BD 双通道主题关键词先验注入，将数据集特定词汇信号以可学习缩放方式融合到边界发射分数。在三个基准数据集上，本方法在 $P_k$、WindowDiff 及边界 F1 指标上均优于强基线，消融实验验证了每项设计的独立贡献。

## 1. 引言

对话话题分割是对话建模研究中的核心前置任务。给定一段多轮对话，DTS 的目标是识别话题切换的边界位置，将对话分解为语义内聚的片段序列，从而为对话摘要 [cite]、多轮问答 [cite]、对话状态追踪 [cite] 等下游任务提供结构化输入。

有监督 DTS 的主流范式以预训练语言模型（如 BERT [cite]）提取话语表示，再经 BiLSTM（+CRF）序列标注预测边界标签。在此范式下，现有工作沿两个维度拓展：其一，以话语对关系分类（UPS [cite]）或对比学习（DyDTS [cite]）增强话题表示的区分度；其二，以 LLM 推理链（Def-DTS [cite]、DMF [cite]）注入显式语义推断能力。然而，上述工作共同存在三方面不足，制约了模型在边界稀疏、多领域场景下的性能上限：

**局限一：词级边界线索缺失。** 现有方法将每条话语压缩为单一句向量后再输入序列模型，丢失了话语内部的词级分布信息。话题切换往往在话语的起止位置体现为词汇选择的突变（如新实体引入、话题结束的程式表达），而这些信号在句级嵌入中被平均稀释。

**局限二：话语重要性无差异建模。** 在一段对话中，不同话语对边界检测的贡献度并不均等：话题引入话语、话题关闭话语所携带的边界信号远强于普通延续性话语。现有方法在 BiLSTM 层面对所有话语等权处理，缺乏自适应聚焦机制。

**局限三：领域词汇先验与神经决策割裂。** 多数方法将关键词、话题词等领域统计视为独立的预处理或后处理步骤，与模型参数分离优化。这一割裂既限制了领域知识对决策的精准作用，也阻碍了关键词信号强度随训练自适应调整。

**本文方法。** 为系统性解决上述三点局限，本文提出 **DUD**。其核心思路是在标准 BiLSTM 有监督分割框架中引入三个结构化模块，分别针对上述三点局限各自建模，并通过统一的端到端目标联合优化。

**本文主要贡献如下：**

- **边界感知双流融合架构。** 并行建立句级语义流（Sentence Stream）与词级边界流（Token Stream）。词流通过 Transformer 编码器 + token LSTM + 边缘门控池化（edge-gated pooling）将话语内 token 序列聚合为话语向量，池化权重以位置偏置函数 $\alpha + (1-\alpha)|2p-1|^\gamma$ 显式放大话语头尾 token 的贡献。两流以可学习标量门 $\sigma(g)$ 动态融合。进一步，将边界建模分解为起始头（start head）与终止头（end head）的联合预测，以 $e_t^{\text{cut}} = 0.5\,e_t^{\text{end}} + 0.5\,e_{t+1}^{\text{start}}$ 显式编码边界前后依赖关系。

- **话语边界信息量自适应加权（UBIW）。** 对序列中每个话语学习标量信息量得分 $w_i \in (0,1)$，以中心化残差放缩方式调制特征：$\mathbf{f}_i \leftarrow \mathbf{f}_i \cdot (1 + \sigma(\theta)\,\tilde{w}_i)$，其中 $\tilde{w}_i = w_i - \bar{w}$ 为序列内中心化权重，$\theta$ 初始化为 $-3$（$\sigma(-3) \approx 0.05$）使模块以近中性状态启动。为捕捉话题边界前后的不对称性，UBIW 分设终止头（end scorer）与起始头（start scorer）两个独立评分器。关键地，两个评分器分别以**边界距离衰减软目标**进行辅助监督：对第 $i$ 条话语，构造目标 $w_i^* = \exp(-d_i / \tau)$，其中 $d_i$ 为其到最近标注边界的距离（终止头对应 end 边界位置，起始头对应 roll 后的 start 位置），温度 $\tau$ 控制衰减速率。该设计将"距边界越近、信息量越高"的归纳偏置显式编码为轻量辅助损失（权重 $\lambda_\text{ubiw}$），与主分割目标联合优化，无需任何额外标注；UBIW 权重在推理阶段可独立读出，具备良好可解释性。

- **Coh/Bnd 双通道主题关键词先验注入。** 从数据集特定的关键词统计中提取两类词汇集合：Coh 通道（cohesion，包含 salient 显著词与 ambient 泛在词，刻画话题内聚性）与 Bnd 通道（boundary，包含 marker 边界标记词并以 core 话题核心词作惩罚项，刻画边界区分度）。每条话语在数据构建阶段被转化为 2-通道匹配分 $\mathbf{k}_t \in \mathbb{R}^2$，在发射层以可学习缩放 $\sigma(\lambda_\text{coh}) \cdot k_{t,\text{coh}} + \sigma(\lambda_\text{bnd}) \cdot k_{t,\text{bnd}}$ 直接偏置正类边界 logit。该机制将无结构领域词表统计升级为可学习的双通道边界先验，与神经表示统一端到端优化，无需任何额外标注。
